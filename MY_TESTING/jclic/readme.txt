Steps to follow in order to install JClic:

- Expand the content of this zip file in a folder
- Create shorcuts for each application, with the following commands:

JClic:
java -jar jclic.jar

JClic author:
java -Xmx256m -jar jclicauthor.jar

JClic reports:
java -jar jclicreports.jar

There are some PNG files, useful for launchers, in the "icons" folder
